Objekte - Eigenschaften und Methoden

Objekte in Javascript sind Datenelemente mit Eigenschaften und oft auch Methoden(Funktionen). Neben vielen vordefinierten Objekten können Sie in Javascript auch eigenen Objekte erzeugen und deren Eigenschaften auslesen und Methoden abrufen.

Ein Objekt funktioniert als eine Zuordnungsliste, die unter bestimmten Namen weitere Unterobjekte, auch Member genannt, speichert. Diese Unterobjekte teilt man in Eigenschaften und Methoden. Methoden sind ausführbare Funktionen, die dem Objekt zugehören, Eigenschaften sind alle nicht ausführbaren Unterobjekte. Sie können als mit dem Objekt verbundene Variablen erklärt werden.

Auf die Eigenschaften eines Objekts können Sie am einfachsten mit der Punkt-Notation zugreifen:

ObjektName.EigenschaftsName

Ein neues Objekt wird mit dem Schlüsselwort new eingeleitet. Sie können für dieses Objekt Eigenschaften deklarieren, in dem Sie ihnen einen Wert zuweisen:

var auto = new Object();
auto.marke = 'Audi';
auto.farbe = 'rot';
auto.baujahr = 2007;
auto.tüv = true;

Scope

Ein Scope beschreibt zu was für Variablen man Zugang hat. Es gibt zwei Arten von Scopes: Globale und Lokale Scopes.

Wenn eine Variable ausserhalb aller Funktionen oder geschweiften Klammen deklariert wird, ist es ein globaler Scope.

Variablen, die nur in einem bestimmten Bereich des Codes verwendbar sind, sind lokale Scopes.

Man sollte immer lokale Variablen nehmen, da bei globalen Variablen eine grosse Chance von Namenskollisionen besteht.

Es gibt zwei Arten von lokalen Scopes: function scope und block scope.

Function scope

Wenn Sie eine Variable definieren, können Sie diese nur innerhalb der Funktion erreichen.

Block scope

Wenn Sie eine Variable mit const oder let in einer geschweiften Klammer definieren, können Sie diese nur innerhalb dieser Klammer ansprechen.

this

Das Schlüsselwort this wird zu einem Wert aufgelöst und zeigt in der Regel auf ein Objekt.

this zeigt standardmässig auf das globale Objekt window. Innerhalb einer Methode, die über objekt.funktion() aufgerufen wird, zeigt es jedoch auf objekt.
