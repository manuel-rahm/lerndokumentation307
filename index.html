Einführung

Javascript ist eine Skript-Sprache und wurde entwickelt, um Benutzerinteraktionen auszuwerten, Inhalte zu verändern, nachzuladen oder zu generieren. Es hat folgende Funktionen:

  • Inhalte Abrufen
    Mit Javascript können Sie alle Elemente, Attribute und Texte einer HTML-Seite abrufen. Beispielsweise können Sie herausfinden, was in das Texteingabefeld mit dem id-Wert email eingegeben wurde.
  • Inhalte ändern
    Mit Javascript könne Sie der Seite Elemente, Attribute und Text hinzufügen. Beispielsweise können Sie die Grösse und Position von Bildern ändern oder nach einem Titel einen Absatz mit Text hinzufügen.
  • Regeln programmieren
    Sie können eine Abfolge von Schritten festlegen, die der Browser ausführen soll, um den Inhalt einer Seite abzurufen oder zu ändern. Beispielsweise kann ein Animationsskript die Abmessungen des Browserfensters prüfen und ein Bild in den sichtbaren Bereich (Viewport) schieben.
  • Auf Ereignisse reagieren
    Sie können festlegen, dass ein Skript nur dann läuft, wenn ein bestimmtes Ereignis eintritt.
    Ereignisse:
    • Benutzer drückt eine Taste.
    • Benutzer klickt auf einen Link.
    • Cursor befindet sich über einem Element.
    • In ein Formular werden Informationen eingegeben.
    • Ein Zeitraum ist verstrichen.
    • Die Webseite ist fertig geladen.
