Variablen

Variablen in PHP starten mit $ und dann kommt der Name der Variablen:

<?php
$txt = "Hello world!";
$x = 5;
$y = 10.5;
?>


Regeln für Variablen:

  • Eine Variable startet mit $.
  • Eine Variable muss mit einem Buchstaben oder _ anfangen.
  • Eine Variable kann nicht mit einer Zahl beginnen.
  • Bei Variablen wird auf Gross- und Kleinschreibung geachtet!

Ausgabetypen

Es gibt hauptsächlich zwei Ausgabetypen in PHP:

  • echo
  • print

echo

Echo gibt einen oder mehrere Strings aus. Echo ist ein wenig schneller als print. Man kann es ohne Klammern verwenden, da es keine Funktion ist.

print

Dieser Befehl ist ähnlich wie echo. Auch hier müssen wir keine Klammern verwenden. Der hauptsächliche Unterschied ist, dass print nur ein Argument zur gleichen Zeit haben kann.
