Variablen

Variablen in PHP starten mit $ und dann kommt der Name der Variablen:

<?php
$txt = "Hello world!";
$x = 5;
$y = 10.5;
?>


Regeln für Variablen:

  • Eine Variable startet mit $.
  • Eine Variable muss mit einem Buchstaben oder _ anfangen.
  • Eine Variable kann nicht mit einer Zahl beginnen.
  • Bei Variablen wird auf Gross- und Kleinschreibung geachtet!

Ausgabetypen

Es gibt hauptsächlich zwei Ausgabetypen in PHP:

  • echo
  • print

echo

Echo gibt einen oder mehrere Strings aus. Echo ist ein wenig schneller als print. Man kann es ohne Klammern verwenden, da es keine Funktion ist.

print

Dieser Befehl ist ähnlich wie echo. Auch hier müssen wir keine Klammern verwenden. Der hauptsächliche Unterschied ist, dass print nur ein Argument zur gleichen Zeit haben kann.

Bedingte Statements

Diese Statements werden normalerweise gebraucht, um Code auszuführen, sobald eine Bedingung zutrifft.

Folgende Statements sind bedingt:

  • if
  • if...else
  • if...elseif...else
    • switch

if

if führt bestimmten Code aus, wenn eine Bedingung zutrifft.

<?php $t = date("H");

if ($t < "20") {
echo "Have a good day!";
}
?>


if...else

if...else führt bestimmten Code aus, wenn eine Bedingung zutrifft und einen anderen, wenn diese Bedingung falsch ist.

<?php
$t = date("H");

if ($t < "20") {
echo "Have a good day!";
} else {
echo "Have a good night!";
}
?>


if...elseif...else

if...elseif...else führt verschiedene Codes für mehr als zwei Bedingungen aus.

if (condition) {
code to be executed if this condition is true;
} elseif (condition) {
code to be executed if this condition is true;
} else {
code to be executed if all conditions are false;
}


Beispiel:

<?php
$t = date("H");

if ($t < "10") {
echo "Have a good morning!";
} elseif ($t < "20") {
echo "Have a good day!";
} else {
echo "Have a good night!";
}
?>


switch

Switch wird verwendet, um verschiedene Aktionen basierend auf verschiedenen Bedingungen auszuführen.

<?php
$favcolor = "red";

switch ($favcolor) {
case "red":
echo "Your favorite color is red!";
break;
case "blue":
echo "Your favorite color is blue!";
break;
case "green":
echo "Your favorite color is green!";
break;
default:
echo "Your favorite color is neither red, blue, nor green!";
}
?>
