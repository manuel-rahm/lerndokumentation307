Syntax

Die Syntax von PHP sieht folgendermassen aus:

<?php
// PHP-code hier rein.
?>


Ein PHP script kann überall im Dokument platziert werden. Ein PHP-Dokument hat die Endung .php

PHP-Dokumente enthalten meist HTML-Elemente und PHP-Code. PHP-Statements enden mit einem Semikolon.

Ein Beispiel für einen einfach Befehl, der Text ausgibt ist:

<!DOCTYPE html>
<html>
<body>

<h1>My first PHP page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>

So würde es als Dokument aussehen.
