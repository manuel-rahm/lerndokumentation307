Einführung

PHP (Hypertext Preprocessor) ist eine serverseitige Skriptsprache. Sie wird verwendet, um Webseiten dynamischer zu machen. Meist wird PHP für Formulare benutzt. Da PHP serverseitig ist, brauchen wir einen Webserver. PHP ist angelehnt an C und Perl. PHP kann auch mit Datenbanken interagieren, um beispielsweise Login-Informationen abzuspeichern.
